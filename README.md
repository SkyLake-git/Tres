# Tres
Uno Dos Tres!
UNOというカードゲームをネットワーク上でプレイできるプログラムです。

クライアント側とサーバー側がありますが、ここで公開しているプログラムはクライアント側です。  

## Requirements
Java version: 8 以上

## 動作環境
Windows [x86] (Recommended)  
MacOS [arm32,arm64]  
Linux [arm32,arm64]  
Windows向けに作成していますが、おそらくこれらのOSでも動作するはずです。

### サポート予定
Browser

### 開発者向け
ビルド情報: `tres.jar/build_info.json`
